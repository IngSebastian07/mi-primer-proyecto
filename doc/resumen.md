## Pasos seguidos para crear un nuevo repositorio ahora en github y enlazarlo con el local.

### A continuación se enumeran los pasos seguidos:

1. **Creación de cuenta en Github:** lo primero que hicimos fue entrar al navegador, escribir Github en la barra de búsqueda y hacer click en el primer resultado. Luego, crear nuestra cuen
ta con un correo electrónico y nombre de usuario, agregando una contraseña para el login por
seguridad.

2. **Creación de un repositorio en Github:** luego, creamos un nuevo repositorio dentro de la p
ágina de Github, agregamos un nombre y lo configuramos de manera pública.

3. **Comandos:** se utilizan los comandos listados y mencionados por el docente para cargar el
repositorio que se encuentra en el dispositivo local en el nuevo repositorio online.

4. **Git push:** en uno de los pasos más importantes, se utiliza el comando git push para sincr
onizar los cambios del repositorio local en la nube.
